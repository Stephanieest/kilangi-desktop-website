## Overview

Kilangi Jewellery is a premium e-commerce website offering customers an elegant and intuitive way to explore high-quality silver and gold jewellery. The website blends modern aesthetics with usability, focusing on visual storytelling, smooth interactions, and accessibility.

Users can:

* Browse jewellery collections including rings, earrings, pendants, bracelets, necklaces, and more
* Discover bestselling and recently viewed products
* Explore curated gift collections by price range and occasion
* Learn about the brand’s ethos and sustainability values
* Read customer testimonials and reviews

---

## Features

### Core Functionality

* **Responsive Design**: Optimized for desktop, tablet, and mobile devices
* **Product Showcase**: Clean product cards with imagery, pricing, and hover effects
* **Category Filtering**: Filter products by jewellery type
* **Gift Collections**: Curated selections for birthdays, anniversaries, and engagements
* **Interactive UI**: Smooth animations, transitions, and hover states
* **Accessibility**: Semantic HTML, ARIA labels, and keyboard-friendly navigation

### Key Sections

1. **Hero Section** – Brand introduction and visual impact
2. **Featured Collections** – Highlighted jewellery categories
3. **Bestsellers** – Popular items with filtering options
4. **Recently Viewed** – Personalized browsing experience
5. **Gift Collections** – Occasion-based jewellery suggestions
6. **Brand Story** – Ethos, craftsmanship, and sustainability
7. **Testimonials** – Customer reviews and social proof

* Browse all jewellery collections

### Run Locally

#### Prerequisites

* Modern web browser (Chrome, Firefox, Safari, Edge)
* Code editor (VS Code recommended)
* Basic knowledge of HTML, CSS, and JavaScript

#### Installation Steps

1. Clone or download the repository to your local machine
2. Open the project folder in your code editor
3. Open `index.html` in your browser

**Optional (VS Code Live Server):**

* Right-click `index.html`
* Select **Open with Live Server**
